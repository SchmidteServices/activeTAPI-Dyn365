Welcome to IM-Service Help Pages

These pages contain the IM-Service documentation for agencies. Help pages contain general concepts and information which is neccessary to work with IM-Service.

The is also a sample application available, written in ASP.NET (C#), which implements all functionality on a demo web-site. Images used in this document are snapshots taken from this demo web-site.

IM-Service connectivity, whitelisting, REST service communication in general are not in scope of this document.
